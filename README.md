# tax-system-api

# 

An API for a dummy tax system with 2 usr roles - accountant and payer

## Installation

``node or docker
npm i 
or
docker build . -t <name>
```

## Usage

```node
npm run dev
or
npm run start and app runs on localhost:8080/api/v1

for tests 
npm run test

docker run <name> start
or
docker run <name> test
```
